#### Запуск приложения
1) Стянуть на локальную машину (git clone https://github.com/damirios/kvartirka_test.git)
2) Установить зависимости:
```
npm i
```
3) Собрать билд
```
npm run build
```
4) Запустить приложение
```
npm start
```
Приложение доступно на http://localhost:3000
